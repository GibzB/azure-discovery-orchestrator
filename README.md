# Azure Discovery Orchestrator

An AI-powered enterprise discovery platform that conducts intelligent cloud consulting sessions using multi-agent reasoning, Azure OpenAI, and Azure AI services.

## What it does
- Runs client discovery conversations (voice + chat)
- Identifies business & technical requirements
- Maps workloads to Azure services
- Generates migration roadmaps
- Produces executive reports & architecture designs

## Architecture
- Azure OpenAI (reasoning engine)
- Azure AI Speech (voice interface)
- Azure AI Search (RAG knowledge base)
- Azure Container Apps (backend runtime)
- Cosmos DB + SQL (state + relational data)
- Azure Front Door (global access)
- Bicep IaC (fully modular infrastructure)

## Core concept
Multi-agent AI system for Azure consulting automation.

## Status
Early-stage architecture and infrastructure setup.
